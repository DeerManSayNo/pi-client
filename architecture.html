📌 关键数据流说明

  1. 用户发消息：ChatInput → POST /api/agent/[id] → startRpcSession() 创建 AgentSession → session.prompt()
  2. SSE 实时流：ChatWindow → GET /api/agent/[id]/events → session.onEvent() → session.subscribe() 推送事件
  3. 会话浏览：SessionSidebar → GET /api/sessions/[id] → lib/session-reader.ts 直接读取 .jsonl 文件（不创建 AgentSession）
  4. Fork 分叉：ChatWindow → send("fork") → 创建新 .jsonl 文件 → 立即销毁旧 wrapper（避免状态污染）
  5. 代码索引：API → code-index/search.ts → SQLite 向量数据库，供 Agent 工具调用时使用
  6. 工具标准化：normalizeToolCalls() 在 session-reader（文件加载）和 ChatWindow（流式事件）两处统一处理字段名映射
